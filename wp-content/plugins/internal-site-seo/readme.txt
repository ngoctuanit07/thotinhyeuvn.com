== Description ==

This WordPress plugin helps improve your websites on-site SEO by creating a web silo where you link back to the category and homepage
from the post and uses tags from other posts to interlinks your posts.

== Installation ==

1. Upload `Internal-Site-SEO.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your settings at Settings > Internal Site SEO Settings tab
4. Add tags to your post as usual and the plugin will automatically interlink your posts.

== Frequently Asked Questions ==

= My links aren't interlinking!

To have interlinking between your posts, you need to add tags to your posts and the plugin will automatically
link to each post whenever that keyword is used in another post.

= What if I don't want certain features of this plugin? =

No problem, you can just turn off the features that you don't like!
